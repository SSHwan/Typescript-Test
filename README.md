# Typescript for Beginners - nomadcoders
타입스크립트 복습

Typescript로 블록체인 만들기

- setting

    1. npm init -y
    2. npm i -D typescript (npm i typescript --save-dev)
    3. npx tsc --init

- tsconfig.json setting

- build -> npm run build
  "scripts": {
    "build": "tsc"
  },

npm i -D ts-node (빌드 없이 타입스크립트 실행)

npm i nodemon (자동 재실행)
